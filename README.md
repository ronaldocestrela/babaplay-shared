# BabaPlayShared

Biblioteca compartilhada para o projeto BabaPlay.

## Estrutura
- **BabaPlayShared.Library**: Contém os modelos, constantes e wrappers utilizados em todo o sistema.

## Instruções
1. Abra a solução `BabaPlayShared.sln` no Visual Studio ou outra IDE .NET.
2. Restaure os pacotes NuGet (`dotnet restore`).
3. Construa o projeto (`dotnet build`).

## Git
Certifique-se de ter `.gitignore` configurado para projetos .NET.
